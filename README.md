# Sokoban

Developed using Python + Pygame

Language: Python

## 说明

+ 自己写的第一个Python小游戏。
+ 编程语言：Python
+ 题目为学校 “计算机基础部” 开设的《Python程序设计》课程的期末大作业。本人并没有选修这门课，但是某个选修了这门课的小学妹在完成这个作业的时候遇到了一些问题。在尽力帮她解决这些问题的同时尝试自己独立完成这个小游戏的开发。
+ 第一次接触Pygame，代码可能写的不是太好
